强调

支持粗体和斜体强调。

斜体

Markdown 预览
这是 *斜体* 强调 这是 斜体 强调
这是 _斜体_ 强调 这是 斜体 强调

句中个别词语使用斜体，使用星号（*）。

粗体

Markdown 预览
这是 **粗体** 强调 这是 粗体 强调
这是 __粗体__ 强调 这是 粗体 强调

句中个别词语使用粗体，使用星号（**）。

粗体和斜体

Markdown 预览
这是 ***粗体*** 强调 这是 粗体 强调
这是 ___粗体___ 强调 这是 粗体 强调

句中个别词语使用斜体和粗体，使用星号（***）。
